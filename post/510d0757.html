字体和文本样式

发表于2024-02-22|更新于2024-03-04|WEB前端
|阅读量:

字体样式

字体设置在网页设计中扮演着关键的角色，它可以直接影响到用户的阅读体验、页面的美观度和整体设计的风格。

字体大小

在CSS中，你可以使用font-size属性来设置元素的字体大小。

取值：

  • px：表示屏幕上的一个点
  • em：相对于父元素的字体大小。如果父元素的字体大小是16像素，那么1.2em就相当于1.2 * 16 = 19.2像素。
  • rem：相对于根元素（html）的字体大小。
  • 百分比：百分比是相对于父元素的字体大小。150%的字体大小是父元素字体大小的150%。

字体系列

在CSS中，font-family属性用于设置元素的字体系列（font family）。字体系列指的是一组字体的排列顺序，如果浏览器无法加载第一个字体，则会尝试加载下一个，依此类推。通过指定字体系列，可以提供备用字体，以确保在不同用户设备上都能良好显示。

在指定字体系列时，可以同时列出多个字体，以提供备用选择。如果字体名称中包含空格，需要使用引号括起来。在实际应用中，建议选择通用的字体系列，以确保在不同设备上都有良好的显示效果。

字体粗细

在CSS中，你可以使用 font-weight 属性来设置文本的字体粗细。该属性接受一些预定义的关键字值，也可以使用数值来表示字体的相对粗细程度。

关键字值：

  • normal: 默认字体粗细。
  • bold: 加粗字体。
  • bolder: 相对于父元素更粗的字体。
  • lighter: 相对于父元素更细的字体。

数值：

  • 数值范围从100到900，以100为间隔，数字越大表示字体越粗。400通常表示普通字体，而700通常表示粗体。需要注意的是，并非所有字体都支持所有粗细的设置。在某些情况下，浏览器可能会根据最接近的可用粗细来渲染文本。

字体样式

font-style 是一个CSS属性，用于设置文本的字体样式，主要用于指定文本是正常、斜体还是倾斜。

行高

行高（line height）是指一行文本的高度，通常用于控制文本行与文本行之间的垂直间距。在CSS中，你可以使用 line-height 属性来设置行高。

line-height 可以使用像素（px）作为单位，表示固定的行高值。

line-height 被设置为固定值 24px，这将使每一行的文本都具有相同的固定高度。这种情况下，无论字体大小如何变化，行高都将保持不变。

当在 CSS 中使用无单位的数字作为 line-height 的值时，它表示相对于当前元素的字体大小的倍数。这个倍数可以是整数或小数。

通过调整line-height，你可以影响文本的可读性和排版效果。合适的行高可以使文本更易于阅读，特别是在长段落中。

font属性

font 是一个CSS的缩写属性，用于同时设置字体相关的多个属性，包括 font-style、font-weight、font-size、line-height 和 font-family。至少需要设置 font-size 和font-family。

文本样式

文本颜色

在CSS中，可以使用color属性来设置文本颜色。

取值：

  • 颜色关键字：black黑色、white白色、red红色、green绿色、blue蓝色、gray或 grey灰色、yellow黄色、orange橙色、purple紫色、pink粉色。
  • RGB值：RGB（Red, Green, Blue）是一种表示颜色的方式，其中红色、绿色和蓝色的组合形成了各种颜色。其中red、green和blue分别是介于0到255之间的整数，表示相应颜色的强度。0表示最小强度（没有颜色），255表示最大强度。
  • RGBA值：如果需要设置颜色的同时还带有透明度，可以使用RGBA值。RGBA值的语法是 rgba(red, green, blue, alpha)，其中alpha是一个介于0到1之间的小数，表示透明度，0表示完全透明，1表示完全不透明。
  • HEX值：HEX（十六进制）值是一种用于表示颜色的方式，通过使用基于十六进制的数字来表示红、绿、蓝三个颜色通道的强度。HEX值的语法是 #RRGGBB，其中RR、GG和 BB分别表示红色、绿色和蓝色通道的强度。每个通道的值由两个十六进制数字表示，范围从00到FF，总共有256个可能的值。#FF0000 表示红色，#008000 表示绿色，#0000FF表示蓝色，#FFFF00表示黄色。通过调整HEX值的组合，可以创建几乎任何颜色。与RGB值类似，如果需要设置颜色的同时还带有透明度，可以使用HEX值的基础上配合RGBA值，通过在HEX值后面添加两个十六进制数字表示透明度。范围是00到FF，其中00表示完全透明，FF表示完全不透明。

文本缩进

在CSS中，你可以使用text-indent属性来设置文本的缩进。该属性指定文本块中第一行的缩进。

文本水平对齐方式

text-align是一个用于设置文本水平对齐方式的CSS属性。

值 描述
left 默认值，左对齐
right 右对齐
center 居中对齐
justify 两端对齐

文本装饰

文本装饰是指在文本上应用一些装饰性效果，例如下划线、删除线、上标、下标等。这些效果可以通过CSS来实现，以下是一些常见的文本装饰属性：

值 描述
none 去除装饰
underline 下划线
overline 上划线
line-through 删除线

text-decoration-color（文本装饰颜色)：用于设置文本装饰的颜色。

text-decoration-style（文本装饰样式)：用于设置文本装饰的样式，可以是 solid（实线，默认）、double（双线）、dotted（点线）等。

字符间距

word-spacing属性用于设置单词之间的额外间距。此属性的值可以是具体的长度值（如像素或 em），也可以是正负值。对中文无效

letter-spacing 属性用于设置字符之间的额外间距。与 word-spacing 不同，letter-spacing 作用于文本的每个字符。

CSS继承性

继承性是CSS中一个重要的概念，指的是子元素可以继承父元素的某些样式属性。这意味着，如果你在父元素上应用了某个样式，这个样式可能会被其子元素继承，并作为子元素的默认样式。

以下是一些常见的 CSS 属性，它们具有继承性：

  • color：定义文本颜色。
  • font-family：定义字体系列。
  • font-size：定义字体大小。
  • font-style：定义字体样式，如斜体。
  • font-weight：定义字体的粗细。
  • line-height：定义行高。
  • text-align：定义文本水平对齐方式。
  • vertical-align：定义行内元素的垂直对齐方式。
